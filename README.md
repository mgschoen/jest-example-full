# Jest Examples

## Setup

Have Node.js installed. Run `npm install` in this repo.

## Usage

``` shell
$ npm test
```

Runs all tests

``` shell
$ npm test -- --verbose
```

Adds per-test results to output

``` shell
$ npm test -- --coverage
```

Outputs code coverage info
